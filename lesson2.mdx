# Μάθημα 2ο: Παράγραφοι, Τίτλοι, Links

## `p` element

Το p element είναι ένα στοιχείο που χρησιμοποιείται για να δημιουργήσει μια παράγραφο.

### Παράδειγμα

```
<p>Αυτό είναι μία παράγραφος.</p>
```

## `h` attributes

Τα h attributes χρησιμοποιούνται για να δηλώσουμε το επίπεδο του τίτλου.

### Παράδειγμα

```
<h1>Αυτός είναι ένας τίτλος.</h1>
```
Τα h attributes χρησιμοποιούνται για να δηλώσουμε το επίπεδο του τίτλου.
Το h1 είναι μεγαλύτερος από το h2, το h2 είναι μεγαλύτερος από το h3 κλπ μέχρι το h6 που είναι ο μικρότερος τίτλος.

## `a` element

Το a element χρησιμοποιείται για να δημιουργήσει ένα link σε μια άλλη σελίδα.

### Παράδειγμα

```
<a href="https://simplifiedcoding.org">simplified Coding</a>
```

Βλέπουμε ότι το link έχει δύο μέρη. 
Το πρώτο μέρος είναι το `href` attribute. Το `href` attribute δηλώνει τη διεύθυνση της σελίδας στην οποία θα μας πάει το link. 
Το δεύτερο μέρος είναι το κείμενο του link. Αυτό είναι το κείμενο που θα εμφανίζεται στη σελίδα.

## `div` element

Με το `div` μπορούμε να χωρίσουμε την ιστοσελίδα μας σε μέρει.
Για παράδειγμα, μια ιστοσελίδα για μια καφετέρια μπορεί να έχει ένα μέρος σχετικά με τον καφέ τους και ένα μέρος με τις τιμές τους.

### Παράδειγμα

```

<div>
    <h2>Ο καφές μας</h2>
</div>

<div>
    <h2>Τιμές</h2>
</div>

```

## `id` attribute

Με το `id` μπορούμε να δώσουμε μια "ταυτότητα" σε ένα `div` ή άλλο element.
Αυτό μπορεί να μας φανεί χρήσιμο αργότερα.

### Παράδειγμα

```

<div id="our-coffee">
    <h2>Ο καφές μας</h2>
</div>

<div id="pricing">
    <h2>Τιμές</h2>
</div>

```

## `class` attribute

Με το `class` μπορούμε να πούμε τι "είδους" είναι ένα element. Μπορείτε να δείτε το παράδειγμα για να καταλάβετε.
Αυτό θα μας φανεί ιδιαίτερα χρήσιμο αργότερα.

### Παράδειγμα

```

<div id="our-coffee" class="header">
    <h2>Ο καφές μας</h2>
</div>

<div id="pricing" class="header">
    <h2>Τιμές</h2>
</div>

```
