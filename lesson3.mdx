# Μάθημα 3ο: Μορφοποίηση κειμένου HTML

## `b` element

Το `<b>` κάνει ένα κομμάτι κειμένου να εμφανίζεται σε έντονη μορφή (bold).

### Παράδειγμα

```
<p>Αυτό είναι <b>ένα κομμάτι κειμένου</b> σε έντονη μορφή.</p>
```

### Αποτέλεσμα

<p>Αυτό είναι <b>ένα κομμάτι κειμένου</b> σε έντονη μορφή.</p>
