# Μάθημα 5ο: Λίστες και Πίνακες

## Λίστες

Στην HTML έχουμε 2 είδη λίστας.

- Μη οργανωμένη λίστα (Unordered list)
- Οργανωμένη λίστα (Ordered list)

### Unordered list

Η μη οργανωμένη λίστα δημιουργείται με την χρήση της ετικέτας `<ul>` και των ετικετών `<li>` για κάθε στοιχείο της λίστας.

```
<ul>
  <li>Στοιχείο 1</li>
  <li>Στοιχείο 2</li>
  <li>Στοιχείο 3</li>
</ul>
```

### Ordered list

Η οργανωμένη λίστα δημιουργείται με την χρήση της ετικέτας `<ol>` και των ετικετών `<li>` για κάθε στοιχείο της λίστας.

```
<ol>
  <li>Στοιχείο 1</li>
  <li>Στοιχείο 2</li>
  <li>Στοιχείο 3</li>
</ol>
```

## Πίνακες (Tables)

Ένας πίνακας στην HTML αποτελείτε από:
- Γραμμές (= rows ή tr) 
- Στήλες (= columns ή th)
- Δεδομένα (= table data ή td)

Ένας πίνακας ξεκινά με το `<table>`. 
Επιπλέον, βάζουμε το `<tr>` για τις γραμμές του πίνακα και το `<th>` για τις στήλες του πίνακα.
Τέλος, προσθέτουμε `<td>` για να προσθέσουμε δεδομένα στον πίνακα.

### Παράδειγμα

```
<table>
  <tr>
    <th>Όνομα</th>
    <th>Ηλικία</th>
    <th>Πόλη</th>
  </tr>
  <tr>
    <td>Γιάννης</td>
    <td>25</td>
    <td>Αθήνα</td>
  </tr>
  <tr>
    <td>Μαρία</td>
    <td>30</td>
    <td>Θεσσαλονίκη</td>
  </tr>
</table>
```
