---

<h2 style="text-align: center;"> `head` tag </h2>

Το ```head``` element είναι το στοιχείο που περιέχει πληροφορίες για την ιστοσελίδα μας.
Περιέχει τον τίτλο της ιστοσελίδας μας, τον τύπο της γλώσσας και άλλα στοιχεία που δεν εμφανίζονται στην ιστοσελίδα μας.

Παράδειγμα:

```html
<head>

</head>
```

---

<h2 style="text-align: center;"> `title` element </h2>

Το ```title``` element είναι το στοιχείο που περιέχει τον τίτλο της ιστοσελίδας μας.
Μπορεί να έχεις παρατηρήσει πως οι ιστοσελίδες έχουν έναν τίτλο στην καρτέλα του browser.
Αυτό γίνεται με τον ορισμό του ```title``` element.

Ας δούμε ένα παράδειγμα:

```html
<head>
    <title> Αυτός είναι ένας τίτλος. </title>
</head>
```

---

<h2 style="text-align: center;"> `style` attributes </h2>

Το ```style``` element είναι το στοιχείο που μας επιτρέπει να εισάγουμε στιλ στην ιστοσελίδα μας.
Θα μάθεις περισσότερες λεπτομέρειες για το styling στο μάθημα 7
και στη σειρά μαθημάτων για τη CSS.

Δε χρειάζεται να μάθεις περισσότερα τώρα. Απλά να ξέρεις πως υπάρχει αυτό το στοιχείο.

Ας δούμε και το παράδειγμα:

```html
<head>
    <style>
        body {
            background-color: red;
        }
    </style>
</head>
```

---

<h2 style="text-align: center;"> `link` element </h2>

To ```link``` element είναι ένα στοιχείο παρόμοιο με το ```style```.
Αντί ο κώδικας για το στιλ να βρίσκεται μέσα στο ```head``` element, είναι σε ένα άλλο αρχείο.
Το ```link``` element μας επιτρέπει να συνδέσουμε το αρχείο με τον κώδικα στιλ με την ιστοσελίδα μας.

Και πάλι, δε χρειάζεται να το μάθεις τώρα.

Ας δούμε άλλο ένα παράπηγμα:

```html
<head>
    <link rel="stylesheet" href="style.css">
</head>
```

---

<h2 style="text-align: center;"> `meta` element </h2>

Το ```meta``` element είναι ένα στοιχείο που περιέχει πληροφορίες για την ιστοσελίδα μας.
Περιέχει τον τύπο της γλώσσας, τον τύπο του encoding και άλλα στοιχεία που δεν εμφανίζονται στην ιστοσελίδα μας.
Χρησιμοποιούμε το ```meta``` ώστε να "ωθήσουμε" την ιστοσελίδα μας να εμφανίζεται σε μηχανές αναζήτησις.

Το ```meta``` μπορεί να περιλαμβάνει:

- ```charset```: Ο τύπος του encoding της ιστοσελίδας μας (Συνήθως `UTF-8`).
- ```name```: Το όνομα του στοιχείου που θα περιγράψουμε.
  - ```keywords```: Λέξεις κλειδιά για την ιστοσελίδα μας.
  - ```description```: Μια περιγραφή για την ιστοσελίδα μας.
  - ```author```: Ο συγγραφέας της ιστοσελίδας μας.
  - ```viewport```: Το viewport της ιστοσελίδας μας, για να την κάνουμε να φαίνεται καλή σε όλες τις συσκευές. (Συνήθως βάζουμε `width=device-width, initial-scale=1.0`).

Ας δούμε ένα παράδειγμα:

```html
<head>
    <meta charset="UTF-8">
    <meta name="keywords" content="HTML, CSS, JavaScript">
    <meta name="description" content="Μια ιστοσελίδα για το HTML.">
    <meta name="author" content="Οδυσσέας Κηρυκόπουλος">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
```

---

<h2 style="text-align: center;"> `script` element </h2>

Το ```script``` element είναι ένα στοιχείο που μας επιτρέπει να εισάγουμε κώδικα JavaScript στην ιστοσελίδα μας.
Μπορεί να μη γνωρίζεις JavaScript ακόμα, όπως και CSS, αλλά μπορείς να συμβουλευτής τη σειρά μας στη JavaScript.
Ξανά, δε χρειάζεται να μάθεις λεπτομέρειες.

Ας δούμε ένα παράδειγμα:

```html
<head>
    <script src="script.js"></script>
</head>
```
