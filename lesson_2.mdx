---

<h2 style="text-align: center;"> `body` tag </h2>

Το ```body``` element είναι το κυρίως στοιχείο της ιστοσελίδας. 
Όλα τα στοιχεία που θέλουμε να εμφανίζονται στην ιστοσελίδα μας πρέπει να βρίσκονται μέσα σε αυτό.

Παράδειγμα:

```html
<body>
    <h1> Αυτό είναι ένας τίτλος. </h1>
    <p> Αυτό είναι ένα κείμενο. </p>
</body>
```

---

<h2 style="text-align: center;"> `head` tag </h2>

Το ```head``` element είναι το στοιχείο που περιέχει πληροφορίες για την ιστοσελίδα μας.
Περιέχει τον τίτλο της ιστοσελίδας μας, τον τύπο της γλώσσας και άλλα στοιχεία που δεν εμφανίζονται στην ιστοσελίδα μας.

Παράδειγμα:

```html
<head>
    <title> Αυτό είναι ένας τίτλος. </title>
</head>
```

---

<h2 style="text-align: center;"> `p` element </h2>

Το ```p``` element είναι ένα στοιχείο που χρησιμοποιείται για να δημιουργήσει μια παράγραφο.

Παράδειγμα:

```html
<p> Αυτό είναι μία παράγραφος. </p>
<p> Αυτό είναι μία νέα παράγραφος. </p>
```

---

<h2 style="text-align: center;"> `h` attributes </h2>

Τα h attributes χρησιμοποιούνται για να δηλώσουμε το επίπεδο του τίτλου.

Παράδειγμα:

```html
<h1> Αυτός είναι ένας τίτλος. </h1>
<h2> Αυτός είναι ένας 2ος τίτλος. </h2>
<h3> Αυτός είναι ένας 3ος τίτλος. </h3>
```

Τα h attributes χρησιμοποιούνται για να δηλώσουμε το επίπεδο του τίτλου.
Το h1 είναι μεγαλύτερος από το h2, το h2 είναι μεγαλύτερος από το h3 κλπ μέχρι το h6 που είναι ο μικρότερος τίτλος.


---

<h2 style="text-align: center;"> `a` element </h2>

Το a element χρησιμοποιείται για να δημιουργήσει ένα link σε μια άλλη σελίδα.

Παράδειγμα:

```html
<a href="https://simplifiedcoding.org"> Simplified Coding </a>
```

Βλέπουμε ότι το link έχει δύο μέρη. 
Το πρώτο μέρος είναι το `href` attribute. Το `href` attribute δηλώνει τη διεύθυνση της σελίδας στην οποία θα μας πάει το link. 
Το δεύτερο μέρος είναι το κείμενο του link. Αυτό είναι το κείμενο που θα εμφανίζεται στη σελίδα.

---

<h2 style="text-align: center;"> `div` element </h2>

Με το `div` μπορούμε να χωρίσουμε την ιστοσελίδα μας σε μέρει.
Για παράδειγμα, μια ιστοσελίδα για μια καφετέρια μπορεί να έχει ένα μέρος σχετικά με τον καφέ τους και ένα μέρος με τις τιμές τους.

Παράδειγμα:

```html
<div>
    <h2>Ο καφές μας</h2>
</div>

<div>
    <h2>Τιμές</h2>
</div>
```

---

<h2 style="text-align: center;"> "id" attribute </h2>

Με το `id` μπορούμε να δώσουμε μια "ταυτότητα" σε ένα `div` ή άλλο element.

Παράδειγμα:

```html
<div id="our-coffee">
    <h2>Ο καφές μας</h2>
</div>

<div id="pricing">
    <h2>Τιμές</h2>
</div>
```



---

<h2 style="text-align: center;"> `class` attribute </h2>

Με το `class` μπορούμε να πούμε τι "είδους" είναι ένα element. Μπορείτε να δείτε το παράδειγμα για να καταλάβετε.

Παράδειγμα:

```html
<div id="our-coffee" class="header">
    <h2>Ο καφές μας</h2>
</div>

<div id="pricing" class="header">
    <h2>Τιμές</h2>
</div>
```
