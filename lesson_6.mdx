<h2 style="text-align: center;"> Λίστες </h2>

Στη HTML έχουμε 2 είδη λίστας.

* Μη οργανωμένη λίστα (Unordered list)
* Οργανωμένη λίστα (Ordered list)

<h3 style="text-align: left;"> Unordered list </h3>

Η μην οργανωμένη λίστα δημιουργείται με τη χρήση της ετικέτας ```<ul>``` και των ετικετών ```<li>``` για κάθε στοιχείο της λίστας.

```html
<ul>
  <li>Στοιχείο 1</li>
  <li>Στοιχείο 2</li>
  <li>Στοιχείο 3</li>
</ul>
```

<h3 style="text-align: left;"> Ordered list </h3>

Η οργανωμένη λίστα δημιουργείται με τη χρήση της ετικέτας ```<ol>``` και των ετικετών ```<li>``` για κάθε στοιχείο της λίστας.

```html
<ol>
  <li>Στοιχείο 1</li>
  <li>Στοιχείο 2</li>
  <li>Στοιχείο 3</li>
</ol>
```

---

<h2 style="text-align: center;"> Πίνακες (Tables) </h2>

Ένας πίνακας στην HTML αποτελείτε από:
* Γραμμές (= rows ή tr) 
* Στήλες (= columns ή th)
* Δεδομένα (= table data ή td)

Ένας πίνακας ξεκινά με το ```<table>```. 
Επιπλέον, βάζουμε το ```<tr>``` για τις γραμμές του πίνακα και το ```<th>``` για τις στήλες του πίνακα.
Τέλος, προσθέτουμε ```<td>``` για να προσθέσουμε δεδομένα στον πίνακα.

Παράδειγμα:

```html
<table>
  <tr>
    <th>Όνομα</th>
    <th>Ηλικία</th>
    <th>Πόλη</th>
  </tr>
  <tr>
    <td>Γιάννης</td>
    <td>25</td>
    <td>Αθήνα</td>
  </tr>
  <tr>
    <td>Μαρία</td>
    <td>30</td>
    <td>Θεσσαλονίκη</td>
  </tr>
</table>
```
